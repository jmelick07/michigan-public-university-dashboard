# Michigan Public University Dashboard

Static dashboard comparing Michigan public university performance metrics and appropriations data.

## Live site

This repository is configured to publish the contents of the `Dashboard/` folder to GitHub Pages.

## Data sources

- IPEDS
- College Scorecard
- Michigan appropriations CSV files provided in this project

## Local files

- `Dashboard/`: static web app
- `MI_Public_University_Combined_Metrics.csv`: combined performance data
- `MI_Public_University_Combined_Metric_Definitions.csv`: metric definitions
- `Appropiations Data/`: appropriations source files
